# MC旧神之史 · 史诗共建计划

> 一个由社区共同编织的《我的世界》史诗长篇故事，一个向所有人开放的创世之书。

欢迎来到 **“MC旧神之史”** 项目！这是一个受 **SCP基金会** 多作者协同创作模式启发的宏大叙事计划。目标是构建一个MC史诗世界观。等我把主线框架搭建完成后，将正式开启故事线投稿
##  投稿方式(注意此仓库为文本仓库，游戏实际插件代码仓库见YURI-PR1ME的javas仓库）

项目主线架构完成后，投稿通道将正式开启。请将作品发送至我的投稿邮箱：

**`[请在此处填入你的投稿邮箱]`**

### 稿件格式要求

1.  **邮件主题**：建议为 `[投稿] - 你的故事副标题`。
2.  **正文/附件内容格式**：
    *   首行必须是 **标题** 与 **记录者（作者）** 信息。
    *   格式模板如下：
标题：MC旧神之史--[你的故事副标题]
----记录者：[你的笔名/署名]

[这里开始你的正文内容...]

text

3.  **投稿形式**：
    *   可直接在邮件正文中撰写（需包含上述格式）。
    *   也可发送附件文件（如 `.txt`, `.docx`），**但文件开头必须包含完整的“标题”和“记录者”信息**。

## ✨ 投稿须知

*   **核心信息**：`标题` 和 `记录者` 为**必需项**，缺失将影响收录。
*   **原创性**：请确保投稿内容为原创。
*   **世界观兼容**：投稿需尊重项目已确立的**基础主线设定**（待公布）。
*   **审阅与回复**：稿件将由我阅读（如果有其他人帮忙就更好了），然后选中会给回复并且添加一个新故事线（mainline之外的分故事线，并不是独立于mainline）。
